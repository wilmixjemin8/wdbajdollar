wdbajdollar

WDBAJ$ is invented in JAS Programming Language at year 2014 by wilmix jemin j.

WDBAJ$ is used in remote database services.

It is a remote database to connect to any program or JAS program or saucer , etc.

It is a NOSQL database.(ie, no need to type sql).

We can perform insertion , deletion, updation, deletion, encryption, and deencryption

by using it.

ADVANTAGES

  1. It saves time and cost

    2.It Provides very high security with encryption

  2. It is a remote database

    4.it is used in remote webapplication

    5.it is used in banking and insurance projects and with all WRIT P.L.

    6.it provides encryption , deencryption ,etc

    7.It is flexible, user friendly, robot , and fast.

    8.We can write only single piece of code to connect to database.

  3. It is also a interactive database.

  4. It is used to store data to - from any database.

  5. we can also store large bulks of data and set the relationship between the table.

  6. It is a platform independant database.

  7. It stores the data in a serialization format.

  8. WDBAJ$ is a NO-SQL Database

  9. We can also executed encrypted class file .dbs file and it will generate a output.

  10. WDBAJ$ is used with Oracle or MYSQL Server .

SYNTAX

<! WDBAJ$ LOGIC !>
